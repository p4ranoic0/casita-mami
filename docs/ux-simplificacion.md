# UX simplificada (marzo 2026)

## Mapa antes → después

Antes: múltiples páginas institucionales y de flujo (`identidad`, `talleres`, `admision`, `familias`, `formulario`, `confirmacion`, etc.).

Después:

1. **Inicio**: propuesta principal + galería + accesos rápidos.
2. **Servicios**: oferta real (nido, guardería, bienestar).
3. **Contacto**: dirección, canales directos y agenda de visita.

## Decisión de interacción

- Se mantuvieron solo 3 rutas visibles en menú.
- Se convirtieron pasos adicionales en modales:
  - Proceso de admisión (Inicio)
  - Talleres/recursos para familias (Inicio)
  - Detalle por servicio (Servicios)
  - Agenda de visita rápida (Contacto)

## Resultado esperado

- Menos carga cognitiva.
- Navegación autoexplicativa.
- Conversión más directa a contacto y visita.
